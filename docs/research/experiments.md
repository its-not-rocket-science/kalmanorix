# Experiments

This document defines the **primary fusion validation path** after the April 2026 audit.

## 1) Concrete benchmark plan

### Benchmark objective
Validate whether Kalman fusion improves retrieval quality over mean fusion on a real mixed-domain benchmark with known query-document relevance labels.

### Datasets (real qrels)
We use three BEIR subsets with explicit qrels:

- `BeIR/nq` → domain label `general_qa`
- `BeIR/scifact` → domain label `biomedical`
- `BeIR/fiqa` → domain label `finance`

Benchmark artifacts are generated by:

```bash
python scripts/build_mixed_benchmark.py
```

Canonical artifact path:

- `benchmarks/mixed_beir_v1.0.0/mixed_benchmark.parquet`

### Specialist models (real domain specialists)
Primary evaluation uses at least three distinct specialist encoders:

- `sentence-transformers/all-mpnet-base-v2` (general QA)
- `emilyalsentzer/Bio_ClinicalBERT` (biomedical)
- `ProsusAI/finbert` (finance)

### Protocol + metrics
Locked protocol implementation:

- `kalmanorix.benchmarks.evaluation_protocol`

Primary metrics:

- Recall@1, Recall@5, Recall@10
- MRR
- nDCG@10

Secondary metrics:

- latency (ms/query)

Comparison:

- `KalmanorixFuser` vs `MeanFuser`

## 2) Code changes required (and implemented)

- Primary benchmark runner: `experiments/run_real_mixed_benchmark.py`
- Benchmark build pipeline: `src/kalmanorix/benchmarks/mixed_domain.py` + `scripts/build_mixed_benchmark.py`
- Validation compatibility entrypoint rewired:
  - `experiments/validate_fusion.py`
  - default mode runs real benchmark
  - `--debug-synthetic` runs toy smoke checks only
- Legacy mixed-domain script demoted to compatibility wrapper:
  - `experiments/mixed_domain_eval.py`
  - default mode now dispatches to real benchmark
  - synthetic mode only via `--debug-synthetic`
- Deprecated synthetic headline script retained as debug-only:
  - `experiments/benchmark_fusion_methods.py`

## 3) New experiment entry points

### Primary headline validation

```bash
python experiments/run_real_mixed_benchmark.py \
  --benchmark-path benchmarks/mixed_beir_v1.0.0/mixed_benchmark.parquet \
  --split test \
  --max-queries 150 \
  --output results/real_mixed_benchmark/real_benchmark_summary.json
```

### Compatibility entrypoints (still real by default)

```bash
python experiments/validate_fusion.py
python experiments/mixed_domain_eval.py
```

### Debug/smoke-only synthetic checks

```bash
python experiments/validate_fusion.py --debug-synthetic
python experiments/mixed_domain_eval.py --debug-synthetic
python experiments/benchmark_fusion_methods.py
```
