# Documentation style guide

## Publication spelling conventions (UK English)

For JOSS-facing and publication-facing prose, use UK English spellings consistently:

- **artefact** (not artifact)
- **organisation** (not organization)
- **standardise/standardised** (not standardize/standardized)
- **optimisation** (not optimization)
- **modelling** (not modeling)
- **behaviour** (not behavior)
- **licence** as a noun, **license** as a verb
- **analyse/analysis** where appropriate
- **programme** only when referring to a planned schedule (software **program** remains acceptable)

Scope note:
- Preserve code identifiers, CLI flags, and file paths exactly as implemented, even when they contain US spelling (for example `results/evidence_registry.json` or `*artifact*` path segments).
